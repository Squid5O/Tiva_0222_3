# Viking Tio(바이킹티오)
-------------------
## 기간
2024년 2월 22일 ~ 3월 25일

-------------------
## 개요
오픈월드 레이드 게임(멀티플레이 자유 컨셉 프로젝트)

-------------------
## 개발
언리얼 블루프린트(5.3v), 언리얼 C++(visual studio)

-------------------
## 기능
- 스팀방 활용한 멀티플레이 (Co-Op) 구현
- 멀티캐스트 이용한 캐릭터 네트워크 연동
- 캐릭터 인벤토리 네트워크 상호작용
- 적 동물 AI FSM 구현 완성
- 맵 & UI 시간 흐름 처리 작업
- 나무 , 돌 , 과일 아이템 파밍 구현
- NPC 및 이벤트 상호작용 구현
